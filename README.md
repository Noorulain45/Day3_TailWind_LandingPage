📌 Project Overview

This landing page includes multiple interconnected pages designed to simulate a real-world website structure. The project demonstrates:
Responsive layout design
Utility-first CSS workflow with Tailwind
Clean and structured HTML
Reusable components
Consistent spacing and typography
Modern UI styling principles

🛠️ Tech Stack
HTML5
Tailwind CSS
PostCSS
Autoprefixer
Node.js (for build process)
